# Exposición: Manejo de Estado (State Management)

Este proyecto es una página web sencilla creada con **HTML, CSS y JavaScript** para explicar el concepto de **Manejo de Estado** en aplicaciones web.  
La página incluye definiciones, ejemplos interactivos y una breve clasificación de los tipos de estado.

---

## 🚀 Objetivo
El propósito de esta página es servir como material de apoyo para una exposición sobre **State Management**, mostrando de manera práctica cómo el estado controla la interacción y la representación de datos en una aplicación.

---

## 📂 Estructura del proyecto
- `index.html` → Página principal con la estructura y contenido.
- `style.css` → Estilos básicos para darle diseño a la página.
- `script.js` → Lógica para el ejemplo interactivo de cambio de estado.

---

## 📖 Contenido
La página incluye:
1. **Definición de Estado**: Qué significa y por qué es importante.
2. **Ejemplo Interactivo**: Un botón que cambia el estado entre *Activo* e *Inactivo*.
3. **Tipos de Estado**:
   - Local
   - Global
   - Persistente
